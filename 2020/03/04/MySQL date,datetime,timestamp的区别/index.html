MySQL date,datetime,timestamp的区别

MySQL有三种日期类型

  • date
  • datetime
  • timestamp

定义

date: 日期 如：2020-03-01 不带时分秒。

datetime: 日期时间 如：2020-03-01 17:20:00 带时分秒。

timestamp: 时间戳， 时间戳是指格林威治时间1970年01月01日00时00分00秒(北京时间1970年01月01日08时00分00秒)起至现在的总秒数。

时间范围

date: ‘1000-01-01’ to ‘9999-12-31’

datetime: ‘1000-01-01 00:00:00’ to ‘9999-12-31 23:59:59’

timestamp: ‘1970-01-01 00:00:01’ UTC to ‘2038-01-19 03:14:07’ UTC

常用方法

查看当前时间戳

两种方法
mysql> select unix_timestamp();
+------------------+
| unix_timestamp() |
+------------------+
|       1583311205 |
+------------------+
1 row in set (0.00 sec)

mysql> select unix_timestamp(now());
+-----------------------+
| unix_timestamp(now()) |
+-----------------------+
|            1583311215 |
+-----------------------+
1 row in set (0.00 sec)

查看当前时间

mysql> select now();
+---------------------+
| now()               |
+---------------------+
| 2020-03-04 16:41:43 |
+---------------------+
1 row in set (0.00 sec)

时间转时间戳

mysql> select unix_timestamp('2020-01-01 00:00:00');
+---------------------------------------+
| unix_timestamp('2020-01-01 00:00:00') |
+---------------------------------------+
|                            1577808000 |
+---------------------------------------+
1 row in set (0.00 sec)

时间戳转时间

mysql> select from_unixtime(1577808000);
+---------------------------+
| from_unixtime(1577808000) |
+---------------------------+
| 2020-01-01 00:00:00       |
+---------------------------+
1 row in set (0.00 sec)

时间戳格式化

mysql> SELECT from_unixtime(1577808000, '%Y年%m月%d日 %H:%i:%S');
+----------------------------------------------------+
| from_unixtime(1577808000, '%Y年%m月%d日 %H:%i:%S') |
+----------------------------------------------------+
| 2020年01月01日 00:00:00                            |
+----------------------------------------------------+
1 row in set (0.00 sec)

时间格式化

mysql> select date_format(now(), '%Y年%m月%d日');
+------------------------------------+
| date_format(now(), '%Y年%m月%d日') |
+------------------------------------+
| 2020年03月04日                     |
+------------------------------------+
1 row in set (0.00 sec)

mysql> select date_format('2020-01-01 12:00:00', '%Y年%m月%d日 %H时%i分%S秒');
+-----------------------------------------------------------------+
| date_format('2020-01-01 12:00:00', '%Y年%m月%d日 %H时%i分%S秒') |
+-----------------------------------------------------------------+
| 2020年01月01日 12时00分00秒                                     |
+-----------------------------------------------------------------+
1 row in set (0.00 sec)
